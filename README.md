# Customer Segmentation Analysis

## Project Overview

This project uses the K-Means Clustering algorithm to divide customers into different groups based on their Annual Income and Spending Score.

## Objective

To identify different customer segments that can help businesses improve marketing strategies and customer engagement.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- VS Code

## Dataset Features

- Customer ID
- Gender
- Age
- Annual Income
- Spending Score

## Project Workflow

1. Load Dataset
2. Data Cleaning
3. Exploratory Data Analysis
4. Elbow Method
5. K-Means Clustering
6. Customer Segmentation
7. Business Insights

## Output

- Age Distribution
- Income Distribution
- Spending Distribution
- Income vs Spending Scatter Plot
- Elbow Method Graph
- Customer Segmentation Graph

## Results

The K-Means algorithm successfully grouped customers into different segments based on their income and spending behavior. These insights can be used for targeted marketing and customer relationship management.
